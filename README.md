# Base scaffold

## Prereqs

+ git
+ node.js
+ bower
+ gulp

## Workflow

1. Pull branch
 * *git checkout [branch]*
 * *git pull*
2. Run gulp
 * *gulp*
3. If necessary, update bower dependencies
 * *bower update*
4. If necessary, edit *gulpfile.js*

## Reminder

* Install extra dependencies via bower
 * *bower install [dependency]* (http://bower.io/docs/api/#install)
